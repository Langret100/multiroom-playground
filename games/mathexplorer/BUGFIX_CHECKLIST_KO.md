# 수학 탐험대 멀티 버그 수정 명단 (점검 대상 리스트)

## A. 전투/공격 (최우선)
- 방장 외 유저 공격 불가 (입력→전송→호스트 판정 경로 전체 점검)
- 직업별 공격 사정거리/판정 불일치 (원본 규칙 vs 호스트 판정 규칙 일치 여부)
- 공격 이펙트/효과음과 실제 판정 분리 문제 (로컬 연출만 실행되는 구조 점검)

## B. 팀 경험치 / 레벨업
- 팀 XP 공유 불안정 (호스트 단일 집계/누락·중복 점검)
- 레벨업 2연속 팝업 (gainExp/조건 체크/UI 재진입 중복)

## C. 레벨업 보상 선택 / 배리어
- 보상 클릭해도 선택 완료 안 됨 (클릭 경로 vs 자동선택 경로 통일)
- 보상 결과는 유저별로 달라야 함 (팀 데이터 / 개인 데이터 분리)
- 선택창 중복/겹침 의심 (phase 재진입, overlay/z-index, pointer-events)
- 선택 후 게임 재개 안 됨 (paused 해제 = 오직 host phase_end 기준)
- 시간초과 자동선택/자동종료 불량 (타임아웃 경로와 클릭 경로 동일 처리)

## D. 보물상자(수학문제/보상)
- 게스트가 상자 먹으면 상자만 사라지고 문제/선택지 미표시 (중요)
- 수학문제 오답 시 무한 반복
- 실패 후 UI 닫혀도 게임 멈춤 유지
- 보물 보상 페이즈도 레벨업과 동일 배리어 규칙 필요

## E. 상태머신 / 동기화 구조 (근본 원인 후보)
- 선택 페이즈 상태 재진입/중복 시작 문제
- UI 상태와 paused 상태가 섞임
- 브리지와 원본 게임 로직 간 중복 제어 충돌

## 수정 전 확정 규칙 (명세)
- 팀 XP/레벨 공유
- 레벨업/보물 보상 결과는 유저별 상이 가능
- 레벨업/보물 화면 표시 중에만 게임 일시정지
- 전원 완료 또는 제한시간 종료 시에만 동시 종료/동시 재개
- 게스트도 공격/상자획득/UI 표시 모두 방장과 동일하게 가능
- 오답/포기/시간초과 구분 (포기·시간초과는 실패 완료 처리 + 배리어 집계)

## 우선순위
1. 진행 막힘 (상자 UI 미표시, 선택 완료 불가, 멈춤 유지, 자동종료 실패)
2. 전투 불가 (게스트 공격, 판정-연출 분리, 직업별 판정)
3. 품질/중복 (이중 레벨업, UI 중복, 상태머신/브리지 구조 정리)
