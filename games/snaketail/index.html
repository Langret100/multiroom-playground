🐍 꼬리잡기 게임 설명

🎮 기본 조작

  • ← / → 방향키 (또는 A/D) — 좌우 회전
  • Shift 또는 ⚡ 버튼 — 부스트 (mass 소모)
  • 마우스/터치 — 클릭한 방향으로 이동

🟡 일반 먹이

  • 크기에 따라 성장량이 다름 (작을수록 많음)
  • 기본 성장량은 작게 설계됨 — 수식으로 빠르게 성장하세요!

🔷 수식 먹이 (육각형)

  • 파란색 = 숫자  |  주황색 = 연산자 (×, ÷, +, −)
  • 맵 곳곳에 완성 가능한 수식이 클러스터로 배치됩니다
  • 먹은 순서가 화면 상단 중앙에 실시간 표시됩니다

✅ 수식 완성 방법

  • 숫자 → 연산자 → 숫자 → 결과 순서로 4개를 먹으면 완성!
  • 예: 2 → × → 3 → 6 을 순서대로 먹으면 계산완료! 🎉
  • 완성 시 5초간 성장 2배 보너스!
  • 순서가 틀리거나 계산이 안 맞으면 재계산! 초기화

⚔️ 충돌 규칙

  • 상대 몸통 앞 80%에 충돌 시 → 작은 쪽이 사망
  • 사망한 뱀은 전부 먹이로 변환됩니다
  • 꼬리 끝 10%는 안전지대

🏆 승리 조건

  • 제한 시간 종료 시 가장 큰 mass를 가진 플레이어 우승
