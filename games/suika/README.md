# 도형게임 (구 수박게임)

## 요약
- Matter.js 기반 물리 합치기 게임(수박게임 스타일)을 **도형 기반**으로 변경한 버전입니다.
- 로비(룸)에서 1:1(또는 CPU)로 플레이할 수 있도록 **임베드(iframe) 실행**을 지원합니다.

## 이번 패치 반영 사항
- 도형 크기(물리 바디/렌더/프리뷰 공통 `size`)를 **1.2배** 키움 (`index.html`의 `SHAPE_SCALE`).
- 도형을 담는 그릇(게임 영역) 높이를 **고정 + 추가 50px 감소**: 디자인 월드 높이 `560 → 510 → 460`.

## 파일 위치
- 실행 페이지: `games/suika/index.html`
- 리소스: `games/suika/assets/`
